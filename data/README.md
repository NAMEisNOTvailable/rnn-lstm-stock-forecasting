# Data

This folder contains the local stock-price files used by the forecasting notebook.

| File | Purpose |
| --- | --- |
| `Google_Stock_Price_Train.csv` | Google training series |
| `Google_Stock_Price_Test.csv` | Google test series |
| `OXY_Stock_Price_Train.xlsx` | Occidental Petroleum training series |
| `OXY_Stock_Price_Test.xlsx` | Occidental Petroleum test series |

These files are third-party market data and remain under the original data-provider terms. See [`../DATA_PROVENANCE.md`](../DATA_PROVENANCE.md) before reusing them.
